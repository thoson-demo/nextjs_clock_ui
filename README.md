# Next.js Clock UI

Digital clock interface built with Next.js and TailwindCSS.

**Demo:** [Clock UI](https://thoson-demo.github.io/nextjs_clock_ui/)  
**Design:** [Dribbble](https://dribbble.com/shots/25338693-Digital-Clock-Simple-UI)

## Getting Started

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000)
